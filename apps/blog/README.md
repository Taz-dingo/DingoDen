# Blog App

`@dingo-den/blog` 是当前优先维护的项目，一个用 Astro 构建的个人博客。

## 开发

在仓库根目录运行：

```bash
pnpm dev:blog
```

或在 `apps/blog` 目录运行：

```bash
pnpm dev
```

## 文档入口

项目文档集中放在 `apps/blog/docs/`：

- `apps/blog/docs/README.md`：文档索引与维护约定
- `apps/blog/docs/spec.md`：设计哲学、产品目标、边界
- `apps/blog/docs/plan.md`：阶段计划与近期方向
- `apps/blog/docs/status.md`：当前状态快照
- `apps/blog/docs/tasks.md`：可执行任务清单
