# Dingo Den Blog

一个使用 Astro 构建的个人博客项目，当前重点是把它打磨成一个安静、克制、content-first 的长期写作站点。

## 开发

在仓库根目录运行：

```bash
pnpm dev
```

构建：

```bash
pnpm build
```

预览：

```bash
pnpm preview
```
